# static-website

This is a static website project that includes HTML, CSS, and JavaScript files to create a simple web application.

## Project Structure

```
static-website
├── src
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # CSS styles for the website
│   ├── js
│   │   └── script.js      # JavaScript for interactive features
│   └── assets
│       └── data.json      # JSON data for dynamic content
├── .gitignore             # Files and directories to ignore by Git
└── README.md              # Project documentation
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd static-website
   ```

3. Open the `src/index.html` file in a web browser to view the website.

## Usage

- The `index.html` file serves as the entry point for the website.
- The `styles.css` file contains all the styling rules for the website.
- The `script.js` file includes JavaScript code for any interactive features.
- The `data.json` file can be used to load dynamic content into the website.
